# whirlpool-client

Java client for [Whirlpool](https://github.com/Samourai-Wallet/Whirlpool) by Samourai-Wallet.

## Build
Build with maven:
```
cd whirlpool-client
mvn clean install -Dmaven.test.skip=true
```

## Java integration
See [JavaExample.java](src/test/java/JavaExample.java) for Java integration.
